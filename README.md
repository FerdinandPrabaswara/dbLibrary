# dbLibrary
dbLibrary merupakan database perpustakaan untuk admin yang bertugas mengatur anggota perpustakaan dan peminjaman buku

Database terdiri dari 12 tabel:
1. anggota
2. buku
3. departemen
4. fakultas
5. penerbit
6. pengarang_buku
7. petugas
8. program_studi
9. seleksi_masuk
10. strata
11. transaksi
12. transaksi_detail
